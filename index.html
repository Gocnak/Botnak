Botnak

Your new best friend for Twitch.tv chat. Botnak is a Java-based IRC chat client with focus on Twitch.tv streams. Created by Gocnak

Features

Commands

Multiple Chat Channels

Supports multiple stream's chat channels in seperate tabs as well as consolidating channels.

Updating

Botnak will automatically check for new updates and let you know.

Troubleshooting

Password and OAuth

The Twitch.tv API as of September 2013 now requires all logins to be made with a OAuth token. For creating a token password to use Botnak, go here.

Twitch Faces

Botnak supports all Twitch icons, including subscriber icons. The face's are downloaded into a seperate folder './Botnak/Face/'. The icons will be unavaiable until Botnak downloads all of the faces and prints out: "Done downloading faces."
